# Roadmap

## Phase 1 ✅ — Foundation

SQLite schema, FTS5 full-text search, 3 MCP tools (remember/recall/forget), 16 unit tests.

## Phase 2 ✅ — Semantic Embeddings

Local all-MiniLM-L6-v2 embeddings, cosine similarity search, hybrid FTS5+semantic ranking, 4 new tests (20 total).

## Phase 3 ✅ — Claude Integration

MCP registration, live cross-session memory persistence confirmed.

## Phase 4 ✅ — Synthesis Tool

`synthesize` MCP tool: cross-memory portrait, change tracking, uncertainty detection, recommendation generation. Includes 4-test suite. Complete as of March 2026. 24 tests total.

## Phase 5 — Open Source

MIT license, GitHub repo, GitHub Actions CI (pytest + mkdocs build), GitHub Pages deployment, PyPI package.
